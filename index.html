PRESET SKILL BUFF

Overall balance adjustments to Preset Skills.
Overall increase except for some Preset Skills.
Since this increase in power will have an outsized impact on the Preset Skill Fixa Termina, we will reduce the power of the Verschmelz series' Everbright Unit Potential accordingly.
*If the Preset Skill Fixa Termina at Level 1 or higher is affixed, it will be adjusted so that it is not weakened by the balance adjustment.Changing specs so that drop-only Preset Skills can also be granted via Enhance Preset Skills.New support item Add Preset Skills Lv. 1 for Enhancing Preset Skills.New exchange items added to Item Recycle.
The following will be added as new exchange items in Item Recycle: N-Rare Drop Rate [1h] +100% and Add Preset Skills Lv. 1.
You can trade N-EX-Cubes for these.
Please note that there is a weekly limit to the number you can exchange for.
